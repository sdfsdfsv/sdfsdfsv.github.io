The paper describes a regularization technique called "Tangent Propagation Training" (TPT) that can mitigate neural collapse and improve the generalization performance and adversarial robustness of deep learning models. TPT involves constraining the tangent space of the last-layer activations to be aligned with the tangent space of the input data manifold. This is achieved by adding a regularization term to the loss function that encourages the last-layer activations to be close to the tangent space of the input data manifold. During training, the tangent space of the input data manifold is estimated using a local linear approximation based on the input data and the current model parameters. The tangent space of the last-layer activations is then constrained to be aligned with this estimated tangent space using a projection operation. The authors show that TPT can induce an underlying mathematical simplicity to the trained deepnet model, which naturally suggests performance benefits. Additionally, they demonstrate that convergence to this rigid structure tends to occur simultaneously with improvements in the network’s generalization performance as well as adversarial robustness.
